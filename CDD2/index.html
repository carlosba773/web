Conociendo a nuestros compañeros

¿Qué vamos a aprender?

Durante las próximas sesiones, trabajaremos los contenidos de estadística de este año. Lo haremos de una forma un poco distinta de la habitual... Esta página web será nuestra hoja de ruta para este tema. En cada clase, accederéis a ella y seguiréis las instrucciones correspondientes a cada día.

¿Cómo lo vamos a aprender?

Siguiendo el orden marcado, esta web os dirá qué hacer en cada momento y cuándo tenéis que entregar cada elaboración.

Como recursos, utilizaremos también el libro de texto y la plataforma Teams para realizar las entregas de cada tarea. También utilizaremos algunas páginas web y Excel, llegado el momento. 

Los materiales necesarios son, además del libro de texto y el ordenador, vuestra libreta de clase y una memoria USB, para guardar toda la información que recopiléis, así como vuestro trabajo.

¿Cómo nos vamos a evaluar?

El objetivo final de esta SdA será que vosotros mismos llevéis a cabo un estudio estadístico. Presentaréis los resultados de dicho estudio mediante dos productos finales: un infografía y un pódcast. Los detalles llegarán en su debido momento. 

También se valorarán determinados productos intermedios, necesarios para llegar a un resultado final adecuado. De nuevo, los detalles se concretarán cuando sea necesario. 

Tened en cuenta que, como siempre, se valorarán los deberes y el trabajo adecuado en el aula, así como vuestra libreta de clase.
